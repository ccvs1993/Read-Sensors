Introduction

The Read Sensors app is a part of small and very simple "eco-system" to make use of today WSN technology. Currently it uses sensd as an Internet gateway (GW) for a local WSN-network. The GW accepts incoming "listeners" on TCP port 1234 (default). The GW redistributes the data received by the GW to all it's listeners. The Read Sensors app acts like listener and view and plots real-time data from the sensor network. WSN data reports are in ASCII text, unix-like, can be handled netcat, gnuplot, grep etc. The current implementation of WSN motes is using Contiki operating system with RIME broadcast network protocol.

Scope

  • Realtime view & plot from remote WSN via GW
  • Plotting scaling of selected data
  • Sharing data and plots using email etc.

With OTG support and a WSN mote

  • View & plot from local WSN via USB mote
  • Be a gateway for local WSN. Sending to proxy
  • Config of local USB mote. (Limited support)
